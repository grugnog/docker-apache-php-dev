Apache PHP DEV Docker image
===========================

A Docker image using [yoshz/apache-php](https://registry.hub.docker.com/u/yoshz/apache-php/) installed with:

* Phing 2.9.1
* Phpunit 4.3.5
* PHP Checkstyle 1.5.5 (including standards for Drupal and Symfony2)
* Ruby 1.9.3
* Rubygems 2.4.4
* Bundler 1.7.7
